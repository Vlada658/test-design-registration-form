# test-design-registration-form
**Описание задания:**  
Разработаны тестовые данные для проверки формы регистрации интернет-магазина с использованием техник тест-дизайна.

**Цель:**  
Проверить валидацию полей "Логин" и "Пароль" с применением классов эквивалентности и граничных значений.
Требования👉 https://rusau.kaiten.ru/p/d/731b641b-545d-4311-a691-c397a21eb1bd

**Что было сделано:**  
- Проанализированы требования к полям "Логин" и "Пароль"  
- Выделены классы эквивалентности:
  - валидные значения  
  - невалидные значения  
- Определены граничные значения (минимальная/максимальная длина)  
- Сформированы тестовые данные для позитивных и негативных сценариев  
- Результаты оформлены в виде таблицы
